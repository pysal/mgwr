    mgwr.utils.truncate_colormap¶

    mgwr.utils.truncate_colormap(cmap: LinearSegmentedColormap, min_val: float = 0.0, max_val: float = 1.0, n: int = 100) → LinearSegmentedColormap[source]¶

    Function to truncate a colormap by selecting a subset of the original colormap’s values

    Args:

    cmap (LinearSegmentedColormap) : Matplotlib colormap to be altered min_val (float) : Minimum value of the original colormap to include in the truncated colormap max_val (float) : Maximum value of the original colormap to include in the truncated colormap n (int) : Number of intervals between the min and max values for the gradient of the truncated colormap

    Returns:

    new_camap (LinearSegmentedColormap) : A new colormap that has been truncated.

    Examples:
    >>> import numpy as np
>>> import matplotlib.pyplot as plt
>>> from mgwr.utils import truncate_colormap
>>> cmap = plt.cm.seismic
>>> cmap = truncate_colormap(cmap, min_val=0.0, max_val=0.5, n=100)
